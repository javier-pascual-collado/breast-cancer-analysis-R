# Breast Cancer Survival Analysis – R

Statistical analysis of clinical factors associated with breast cancer relapse time
in a cohort of 180 patients.

## Methods
- Univariate descriptive analysis (mean, SD, variance, percentiles)
- Spearman correlation between all variable pairs
- T-test (menopausal status vs quantitative variables)
- ANOVA (histological grade vs quantitative variables)
- Multiple linear regression (relapse time prediction)

## Key findings
Histological grade and number of positive lymph nodes are the strongest
independent predictors of relapse time (R² = 32.12%).

## Technologies
- R
- Dataset: 180 breast cancer patients
